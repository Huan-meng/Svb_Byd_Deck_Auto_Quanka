# 影之诗（Shadowverse WB）全卡图爬取脚本

本项目包含一个 Python 脚本，用于自动爬取《影之诗》（Shadowverse World Beyond）国服官网上的全部卡牌图片。

脚本会遍历所有职业，自动下载卡牌的基础卡图、进化后卡图，以及特殊的 **异画（Token / Style）** 版本，并将其统一转换为体积更小、支持透明通道的 `WebP` 格式保存。

## ✨ 特性

- **全职业覆盖**：自动爬取中立、妖精、皇家、法师、龙族、梦魇、主教、超越者 8 个职业的卡牌。
- **多形态卡图**：
  - **基础卡图**：普通卡牌原画。
  - **进化卡图**：标记为 `_evo` 的强化版原画。
  - **异画 Token**：特殊版本的插画，包括联动皮肤等。
- **智能识别爆能强化**：自动解析卡牌技能文本中的“爆能强化”信息，体现在文件名中（例如 `5@10` 表示 5 费卡牌，爆能后为 10 费）。
- **格式转换**：自动将下载的 PNG 图片转换为 `WebP` 格式（quality=90），在保持高画质的同时大幅减小文件体积。
- **断点续爬**：已存在的文件会自动跳过，方便增量更新。

## 🛠️ 依赖环境

脚本需要 Python 3.6+ 环境，并依赖以下第三方库：

- `requests`：用于发送 HTTP 请求。
- `Pillow`：用于图片格式转换（PNG 转 WebP）。

**安装依赖：**

```bash
pip install requests Pillow
```

## 🚀 使用方法

直接运行主脚本即可：

```bash
python 全卡爬取（含爆能 token 异画）.py
```

运行后，脚本会在当前目录下自动创建 `SV_WB_Cards` 文件夹，并按职业分类存放下载的卡图。

## 📂 目录结构示例

```
SV_WB_Cards/
├── 中立/
│   ├── 1_10201110_1_1.webp      # 基础图 (费用_卡ID_攻击_生命)
│   ├── 1_10201110_evo.webp      # 进化图
│   ├── 2@4_10001110@1_2_2.webp  # 异画1 (带爆能标记)
│   └── ...
├── 妖精/
├── 皇家/
└── ...
```

## ⚙️ 核心逻辑说明

- **数据源**：通过官方公开接口 `https://shadowverse-wb.com/web/CardList/cardList` 获取卡牌元数据。
- **命名规则**：为避免特殊字符导致的兼容性问题，文件名主要使用 **卡牌 ID** 作为主键，辅以 **费用**、**攻击/生命值** 和 **是否进化/异画** 等属性进行命名。
- **请求频率**：内置 1 秒延迟 (`time.sleep(1)`)，保护服务器并避免被封禁。
